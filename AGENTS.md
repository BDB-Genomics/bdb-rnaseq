## 🔍 Codebase Exploration Skills

You have access to **Understand-Anything** agent tools in this environment:
* `understand`: Scans code structures to build `.understand-anything/knowledge-graph.json`.
* `understand-dashboard`: Starts the React dashboard.
* `understand-chat`: Query the codebase knowledge graph.
* `understand-diff` / `understand-explain`: Inspect changes and individual modules.
